## Overview

**CE-rent** is a full-stack web application designed for renting camping equipment. It provides a complete e-commerce experience including product browsing, cart management, and secure checkout, backed by a robust administrative dashboard.

## Features

- **Product Catalog** - Browse and search a wide variety of camping gear.
- **Cart** - Easily manage items and quantities with dynamic updates.
- **Secure Checkout** - Integrated **Midtrans** payment gateway for safe transactions.
- **User Dashboard** - Track order history and manage profile settings.
- **Admin Portal** - Comprehensive tools for managing inventory, orders, and transactions.
- **Reviews** - User testimonial system.

## Tech Stack

- **Backend** - Laravel 11.x, PHP 8.2+
- **Frontend** - Livewire 3.x, Tailwind CSS, Alpine.js
- **Database** - MySQL
- **Payment** - Midtrans Gateway
